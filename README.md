# 藍圖印刷 LANTU

### 模擬印刷電商網站

### [DEMO 頁面](https://shuhuiyun.github.io/LANTU/#/ '藍圖印刷')

模擬電商網站，使用者可以瀏覽商品、最新消息、查詢商品訂單。後台則提供優惠券、文章、商品新增修改及刪除等功能。

## 網站功能

- 前台

  - 文章列表可點及 Tag 分類
  - 瀏覽商品清單
  - 單一商品頁提供隨機商品建議
  - 輸入信箱可查詢訂單

- 後台
  - 新增修改商品、文章、折價券
  - 切換上、下架
  - 查看客戶訂單
